使い方

  1. メインメニューから比較したいCLIライブラリを選択します
  2. 選択したライブラリのターミナル画面が表示されます
  3. コマンドを入力して各ライブラリの特徴を体験できます
  4. メインメニューに戻って他のライブラリと比較できます

対応ライブラリ

Python

  • Click - デコレーターベースのコマンド定義
  • Typer - 型ヒント活用とリッチ出力
  • Fire - 自動CLI生成

Node.js

  • Commander.js - チェーン可能なAPI
  • Yargs - 複雑なオプション処理
  • Inquirer.js - インタラクティブプロンプト

その他の言語

  • Cobra (Go) - サブコマンド構造
  • Clap (Rust) - 強力な引数解析
  • Thor (Ruby) - Rakeライクなタスク定義
  • OptionParser (Ruby) - シンプルなオプション解析

基本コマンド

--help ヘルプを表示
hello --name Your-Name 基本的な挨拶コマンド
info ライブラリ情報を表示
